
## Notes for Claude Code

- Render the landing as a single Next.js page, sections stacked, with sticky CTA button in top-right after first scroll.
- Mobile-first — headline should work at 380px width (line-wrap deliberately controlled with `<br />` tags where specified).
- The three-card pricing block uses the same visual language as the in-app Readiness Card, for consistency.
- Lead the hero with whitespace, not imagery. No stock photos of doctors. No AI-generated hero visuals. The copy is the visual.
- Final CTA on every section scrolls back to hero CTA by default; sticky top button also visible after 300px scroll.
- Animation: fade-in on scroll, nothing more. No parallax. No background gradients. No hero background video.
